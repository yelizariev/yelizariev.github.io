Sometime odoo users want to have usual "Sent" mail box.

To implement it I've added new computed field:

{% highlight python %} self.sent = len(self.notified_partner_ids) > 1 \ or len(self.notified_partner_ids)==1 \ and self.notified_partner_ids[0].id != self.author_id.id {% endhighlight %}

This field is need to filter out some messages (notifications) which don't have recepients.

This menu shows messages in threads (like in other menus), but shows only sent messages:
